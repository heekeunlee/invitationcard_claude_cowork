# 결혼식 청첩장 디자인 비교 분석 & 셀프 제작 툴

한국 모바일/종이 청첩장 시장을 비교 분석하고, 직접 모바일 청첩장을 만들 수 있는 인터랙티브 대시보드입니다.

## 구성

**📊 비교 분석 탭**
- 전달 방식별(모바일·종이·셀프) 검색 트렌드 (네이버 데이터랩 2023.06–2026.06)
- 유형별 다차원 레이더 비교, 가격대 분포
- 필터링 카드 비교 + 정렬 가능한 비교표 (8개 디자인 스타일)

**✏️ 셀프 제작 툴 탭**
- 템플릿 8종 + 색상 테마 선택
- 메인 정보·인사말·예식 안내·오시는 길·계좌·RSVP 입력 → 실시간 모바일 미리보기
- **대표 이미지 표현 방식**: 실제 사진 / 일러스트풍 / 어반스케치 / 수채화 / 라인아트 / 플라워 모티프(사진 없이)
- 내보내기: PNG 이미지 · 단독 HTML 파일 · 카카오톡 공유 문구 복사

## 사용 방법

`index.html`을 브라우저에서 열면 바로 사용할 수 있습니다. (별도 빌드 불필요)

## 데이터 출처

네이버 데이터랩 검색 트렌드, 네이버 쇼핑 실매물 가격, 블로그 후기 정성 분석 (2026.06 기준).
점수형 지표는 후기·사양 기반 추정치로 실제 업체·상품별 편차가 있습니다.
